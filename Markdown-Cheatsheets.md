# 📝 Markdown Basic Syntax Cheat Sheet

## 🏗️ Structure
- **Headings:** Use 1-6 `#` symbols followed by a space.
- **Paragraphs:** Separate text with a blank line.
- **Line Breaks:** End a line with two or more spaces and then hit return.
- **Horizontal Rules:** Use three or more asterisks `***`, dashes `---`, or underscores `___`.

## 🎨 Emphasis
- **Bold:** `**text**` or `__text__`
- **Italic:** `*text*` or `_text_`
- **Bold & Italic:** `***text***`
- **Strikethrough:** `~~text~~`

## 📋 Lists
- **Unordered:** Use `-`, `*`, or `+` followed by a space.
- **Ordered:** Use numbers followed by a period (e.g., `1. `, `2. `).
- **Task Lists:** `- [x]` for complete, `- [ ]` for incomplete.

## 🔗 Links & Images
- **Links:** `[Display Text](https://url.com)`
- **Images:** `![Alt Text](path/to/img.jpg)`

## 💻 Code
- **Inline Code:** Enclose text in single backticks: `` `code` ``
- **Code Blocks:** Enclose lines in triple backticks:
    ```bash
    echo "Hello World"
    ```

## 📜 Others
- **Blockquotes:** Use `>` before the text.
- **Escaping:** Use a backslash `\` to show literal characters (e.g., `\*`).
